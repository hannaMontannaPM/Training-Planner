# Training Planner

## Автор
Андреюк Александр 

## Описание
Данное приложение позволяет планировать тренировки, 
добавлять записи с датой, типом и длительностью, фильтровать 
их и сохранять/загружать данные в формате JSON.

## Использование
1. Введите дату (ГГГГ-ММ-ДД), тип тренировки и длительность.
2. Нажмите "Добавить тренировку" и данные появятся в таблице.
3. Для фильтрации выберите тип и/или введите дату.
4. Нажмите "Сохранить в JSON" для сохранения данных.
5. Нажмите "Загрузить из JSON" для загрузки сохраненных данных.
